Class ExcludeCoursesInSIS

Exclude(Student, CareerProgPlanSubpln, DateTime, List<(ADVLocationRequirementLine Location, List<Int32> coursesToExclude)>, String)

Exclude courses in SIS that according to SPA should not be there, for a particular student and programme-plan. If an advisement override for the location already exists, exclusions will be added to it and in that case some of the below parameters are ignored (requirementDate and excludeReason).
