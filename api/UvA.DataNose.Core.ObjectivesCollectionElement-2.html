Class ObjectivesCollectionElement<T1, T2>

Holds a collection of items of type T1, but items of another type T2 can be added to it with a ModalObjectPicker by first converting them to T1 with the addToTargetAction (in this case T2 = Objective is converted to a T1 = ObjectiveLink)
